# StockBuzzTracker
实时追踪 Reddit、Twitter热门讨论股票
社交平台过去 24 小时内提及次数最多的美股，计算热度变化并展示排名。

## ✨ 功能
- 📊 热度排名表：展示股票代码、提及次数、24h变化、点赞数
- 📈 趋势图表：提及次数随时间变化折线图
- 🌐 多平台聚合：Reddit、Twitter、Seeking Alpha、富途牛牛、长桥
- ⏰ 实时更新：后端每小时自动抓取，前端每5分钟刷新
- 🎨 深色/浅色主题
